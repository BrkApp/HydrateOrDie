## Format des Stories

Chaque fichier de story suit le template suivant:

- **Epic** et **Story ID**
- **Status** (Not Started, In Progress, Completed)
- **Priority** (Critical, High, Medium, Low)
- **Estimated Effort** (en heures)
- **User Story** (As a... I want... so that...)
- **Acceptance Criteria** (liste numérotée)
- **Technical Notes** (emplacements fichiers, packages requis)
- **Dependencies** (stories dont cette story dépend)
- **Definition of Done** (checklist de validation)
- **Links** (vers epic parent, story précédente, story suivante)
